# Scripts 目录

这个目录包含项目的辅助脚本。

## generate-mock-data.cjs

**功能**：生成完整的分析报告 Mock 数据

**用途**：
- 为所有企业（E001、E002、E003）生成数据
- 为所有时间范围（month、quarter、year、all）生成数据
- 为所有模块（信用评分、风险分析、资金流向、交易趋势、行业对比、AI分析）生成数据

**运行方式**：
```bash
npm run generate:mock
# 或者
node scripts/generate-mock-data.cjs
```

**输出**：
- 文件：`src/mocks/data/analysisReport.json`
- 数据条目：3 企业 × 4 时间范围 × 6 模块 = 72 条数据

**何时使用**：
- 添加新的企业时
- 修改数据生成逻辑后需要更新静态数据
- 初始化或重置 mock 数据

**数据生成逻辑**：
1. 基于 E001（华夏供应链科技有限公司，AAA 级）的基准数据
2. E002（东方物流集团，AA 级）：使用 0.85 倍数
3. E003（盛世贸易有限公司，A 级）：使用 0.72 倍数
4. 确保数据逻辑关联：
   - 信用评分 ↑ → 风险 ↓
   - 信用评分 ↑ → 还款率 ↑
   - 信用评分 ↑ → 行业排名 ↑

**注意事项**：
- 运行此脚本会覆盖现有的 `analysisReport.json` 文件
- 请确保在覆盖前备份重要的自定义数据
- 生成的数据会自动进行范围校验（如信用评分 300-1000，还款率 0-100%）
